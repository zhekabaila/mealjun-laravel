### Prompt 3.1 — Create CloudinaryService

```
Buat file app/Services/CloudinaryService.php dengan:

Method __construct():

Method uploadBase64($base64Image, $folder, $publicId = null):

- Throw Exception jika response failed

Method delete($publicId):
- Build signature dengan sha1(public_id + timestamp + apiSecret)
- POST ke Cloudinary destroy endpoint
- Return bool (true jika result = 'ok')

Reference: INSTRUCTIONS.md section 3 (Service Classes)

Key points:
- Input HARUS berformat 'data:image/type;base64,...'
- Folder names: mealjun/products, mealjun/avatars, mealjun/gallery, mealjun/about
- Return array consistency
- Exception handling
```

### Prompt 3.2 — Create EvolutionApiService

```
Buat file app/Services/EvolutionApiService.php dengan:

Method __construct():
- Inject config('services.evolution.*') ke protected properties

Method checkWhatsappNumbers($numbers):
- POST ke {baseUrl}/chat/whatsappNumbers/{instanceName}
- Headers: Content-Type application/json, apikey
- Payload: { 'numbers': [...] }
- Return array dari response
- Throw Exception jika failed

Method sendText($number, $text):
- POST ke {baseUrl}/message/sendText/{instanceName}
- Headers: same
- Payload: { 'number', 'text' }
- Return response.json()
- Throw Exception jika failed

Method formatNumber($number):
- Private/protected method
- Regex: hapus non-digit character
- If start with '08': convert to '62' + substr(1)
- Else: return as-is
- Example: '081313747177' → '6281313747177'

Method notifyAdmin($adminNumber, $message):
- Try block:
  1. Format nomor via formatNumber()
  2. Check validitas via checkWhatsappNumbers()
  3. Verify response 'exists' field
  4. Jika valid: sendText()
  5. Return true
- Catch block: Log error via \Log::error(), return false
- PENTING: Jangan throw exception (harus silent fail)

Reference: INSTRUCTIONS.md section 3 (Service Classes)

Key points:
- Nomor format: 628xx (international, no +)
- formatNumber() auto-convert '08xx' ke '628xx'
- notifyAdmin() check validitas SEBELUM send
- Error handling: SILENT (log tapi return false, jangan stop proses)
- All calls via Evolution API
```

**Expected Output**:

- ✅ CloudinaryService.php di app/Services/
- ✅ EvolutionApiService.php di app/Services/
- ✅ Both services injectable via constructor
- ✅ All methods documented dengan PHPDoc

### Prompt 4.1 — Configure Sanctum Middleware

````
Setup Sanctum middleware di bootstrap/app.php:

       $middleware->statefulApi();
````

2. Verify di routes/api.php:
    - Auth routes (tanpa middleware): POST /api/auth/login
    - Protected routes: middleware('auth:sanctum')

3. Test via artisan:
   php artisan route:list --path=api

Reference: INSTRUCTIONS.md section 8 (Authentication)

```

### Prompt 4.2 — Create AuthController

```

Buat file app/Http/Controllers/Api/AuthController.php dengan:

Method login(Request $request):

- Validate: email (required|email), password (required|string)
- Query User: where email + where is_active = true
- Check password via Hash::check() vs password_hash field
- If fail: throw ValidationException with custom message
- If success:
    1. Update user.last_login = now()
    2. Create token: $user->createToken('api-token')->plainTextToken
    3. Return json: message, token, user (id, email, full_name, role)
    4. Status: 200 (implicit)

Method logout(Request $request):

- Delete current token: $request->user()->currentAccessToken()->delete()
- Return json: message "Logout berhasil"
- Status: 200

Method me(Request $request):

- Return json: $request->user() (full user object)
- Status: 200

Reference: INSTRUCTIONS.md section 7 (Auth API)

Key points:

- Password field di DB namanya password_hash (bukan password)
- Use Hash::check() untuk compare
- Token di Sanctum auto-stored
- me() returns full user object
- All messages dalam bahasa Indonesia

```

### Prompt 4.3 — Add Auth Routes

```

Update routes/api.php dengan auth routes (tanpa middleware):

```php

    Route::prefix('auth')->group(function () {
        Route::post('/logout', [AuthController::class, 'logout']);
        Route::get('/me', [AuthController::class, 'me']);
    });
});
```
